Static Web Apps CLI (Preview)

A Local Development Tool For Azure Static Web Apps

Try the SWA CLI Quickstart ↖️

Install SWA CLI

Get started by installing the Azure Static Web Apps CLI using yarn or npm.

Configure SWA CLI

Explore the environment variables and config settings available for SWA CLI.

Login to Azure

Use the swa login command to authenticate with the relevant services.

Initialize SWA

Use the swa init command to setup a project with a preferred web technology.

Develop SWA (local)

Use the swa start command to preview and debug your SWA on a local server.

Deploy SWA (cloud)

Use the swa deploy command to build and deploy SWA to multiple environments.
